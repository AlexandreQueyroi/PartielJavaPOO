**LE SUJET**  
Je vous ai fourni le code source Java d’une application permettant de simuler un système de gestion de camping avec la location d’emplacements de tentes et de Mobil-homes : 

* 2 types de location possible : emplacement de tente ou Mobil-home  
* Plusieurs tailles d’emplacement possibles (60m2, 100m2, 150m2 et 200m2) à des tarifs journaliers respectivement de 10€, 20€, 30€ et 40€  
* 2 types de Mobil-home : le petit pour 4 personnes max à 30€/jour et le grand pour 6 personnes max à 50€/jour

Cette application doit fonctionner en mode console.

NE PAS supprimer les TODO ou TODO QUESTION du code source.
